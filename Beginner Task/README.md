🏠 Boston House Price Prediction using Machine Learning 

This project focuses on predicting Boston house prices using Machine Learning regression techniques.
The dataset contains various housing-related features such as crime rate, number of rooms, tax rate, pupil-teacher ratio, and more.

The objective of this project is to build an accurate regression model capable of estimating house prices based on these factors.

🚀 Features
1. Data preprocessing and cleaning
2. Handling missing values
3. Exploratory Data Analysis (EDA)
4. Correlation heatmap visualization
5. Model training using: Linear Regression & Random Forest Regressor
6. Performance evaluation using multiple regression metrics
7. Feature importance analysis
8. Visualization of actual vs predicted prices
9. Model serialization using Pickle

🛠️ Technologies Used
1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
6. Scikit-learn
7. Pickle

📈 Visualizations Included
1. Correlation Heatmap
2. Actual vs Predicted Price Graph
3. Feature Importance Graph
